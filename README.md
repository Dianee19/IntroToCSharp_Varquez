# Task Description
This program is a console-based C# application that tracks:
Weekly fuel budget
5-day fuel expenses
Total distance traveled
Fuel efficiency rating
Budget status.

The system performs input validation and generates a formatted weekly audit report.

# How to Use

1. Open the project in Visual Studio.
2. Run the program
3. Enter the required information:
   - Driver's full name
   - Weekly fuel budget
   - Total distance traveled
   - Daily fuel expenses for 5 days
4. The system will automatically:
   - Validate the distance input
   - Compute total fuel spent
   - Calculate average daily expense
   - Determine fuel efficiency rating
   - Check if the expenses are within budget
5. The program will display a formatted Weekly Audit Report in the console.
